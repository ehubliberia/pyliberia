ABOUT


The Python Users Nigeria Group (Python Nigeria) is a body that encompasses all friends of the Python programming language in the Nigerian environ.

The community is open to everyone at all levels. So whether you are resident in Nigeria or outside the country, an absolute beginner or a maestro, program in Python or other languages but have some interest in Python, our community will embrace you. We have an active Slack team, where we engage ourselves regularly. You can check out our Github page for projects we have or are working on.
